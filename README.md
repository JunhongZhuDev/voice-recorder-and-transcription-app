# Voice Recorder & Transcription App

This project is developed based on, just add a transcription function:

- 🎤 Record your voice directly in the browser
- 🔊 Play back your recordings
- 📝 Convert your recordings to text using OpenAI Whisper
- 💻 View transcriptions in real-time

---

## Features

- Record audio using your computer microphone
- Playback recorded audio
- Upload `.webm` audio to the backend
- Automatic transcription using Whisper AI
- Optional: Translate audio to English (Whisper supports multiple languages)
- Simple, responsive frontend
